So, you want to know about mini.css, right?

I'll give you the elevator pitch and then you can decide if you want to stay or leave. So, mini.css is a CSS framework (yes, another one) that tries to bridge the gap between micro-frameworks and full-featured UI toolkits. It's similar to things like Bootstrap, Semantic UI, Foundation etc, but it's a lot lighter, more modern (based on Flexbox) and easily customizable. It's responsive, it doesn't use any Javascript, it's actively maintained, what else is there that you could possibly ask for? Oh, did I mention it has some default themes you can try out without getting your hands dirty? Yes, we call them flavors!

What's with the project? Good question...
